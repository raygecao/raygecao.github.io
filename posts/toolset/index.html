工欲善其事，必先利其器。

Unix 通用配置

  • Oh My Zsh：专业的 zsh 管理工具，有丰富的插件、主题及好用的 git alias 等。
  • agnoster-zsh-theme：好用的 zsh theme，基于历史记录的命令行即时提示。
  • fzf：命令行模糊搜索工具，UI 展示出模糊搜索到的历史记录。
  • mosh：移动的 shell，增强型 ssh。支持间歇式连接，可以避免 ssh 网络切换/超时导致 session 断开的问题。
  • tmux：终端复用器，在单个 shell 终端上管理多个程序 session，配合 mosh 使用更香。
  • tig：git 的文本模式接口，交互性与易用性大大提升。
  • sshuttle：ssh 透明代理，浏览器访问专网时时无需设置端口转发，连洞神器。

容器相关工具

  • dive：镜像的 layer 可视化分析工具，展示各层内容以便优化镜像大小。
  • crane：镜像的管理工具，主要用来与 registry 进行交互。
  • k9s：k8s 可视化管理工具。

多媒体相关

  • snipaste：可以悬浮铺开的截图工具。
  • sqoosh：图片压缩工具，写文档/博客必备。
  • licecap：捕获桌面动作的 gif 生成器，写文档/博客必备。
  • 图表
    • excalidraw：在线手绘风绘图工具。
    • drawio：在线流程图绘制工具。
    • processon：在线制作流程图与思维导图。
    • D2：可编程的声明式图表。
  • 图标库
    • fontawesome
    • iconfont
    • heroicons
    • terrastruct
    • mdi icon
  • 图片库
    • freepik
    • unsplash
  • 网页设计
    • onepagelove
    • screenlane

效率工具

  • codimd：可协作的实时 markdown 编辑器，需私有化部署，支持 markdown slide 制作。
  • Alfred：流行的 MAC 搜索工具，可以集成很多实用小工具，如单位转换、翻译、搜索引擎等。
  • Navicat：数据库管理 GUI。
  • ByteBase：开源的数据库 DevOps 工具，使数据库与 VCS 集成，实现 Database as Code。
  • 正则表达式
    • 正则表达式手册
    • regexper：可视化展示正则表达式。
    • rubular：正则表达式匹配校验器。
  • carbon：代码图片化工具。
  • dot-to-ascii：基于字符的流程标识，常用于高大上的代码注释中。
  • replit：在线 IDE，丰富的代码提示虐杀 go playground。
  • Reactive Resume：开源的简历制作工具。

网站相关

  • vercel：个人网页的部署工具。
  • github pages：静态网页的部署工具。
  • next.js：易上手的的 React web 应用框架。结合 vercel、nextAuth.js 一站式建站。
  • hugo：静态网页生成器，搭建个人博客不二之选。
